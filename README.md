# LifeVault

This is a basic Next.js dashboard project with Tailwind CSS.

## Available Scripts

- `npm install` – install dependencies
- `npm run dev` – start development server on http://localhost:3000
- `npm run build` – build for production
- `npm run start` – start production server

The main dashboard page (`pages/index.tsx`) includes:

- A search bar
- An "Upload Document" button

Tailwind CSS is configured via `tailwind.config.js` and imported in `styles/globals.css`.
